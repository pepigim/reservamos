## Instrucciones para correr el proyecto de Pronostico Clima ##

- Clonar el repositorio reservamos: git@github.com:pepigim/reservamos.git
- Instalar node.js: https://nodejs.org/en/
- Luego de clonar en la consola de tu sistema operativo dirigirse a la carpeta clonada.
- Ejecutar el commando: npm install
- Despues ejecutar: npm start
- Se escribe el nombre de la ciudad a buscar, se hace clic en el botón buscar y se despliegan 
  abajo los resultados y se hace clic sobre cualquier Ciudad y se despliega un modal con el clima
  de los 7 días de la ciudad seleccionada.

OBSERVACIONES: Se utilizo la ultima versión de material ui para el desarrollo del challenge
